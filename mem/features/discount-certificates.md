type: feature

Платформа має систему сертифікатів знижок для послуг фото/відео/музика, керовану з адмін-панелі (вкладка «Сертифікати»).

**Модель даних** (`user_certificates`):
- Один сертифікат на користувача (UNIQUE user_id)
- `is_active` — вмикач адміна
- `discount_type` — 'fixed' (USD) або 'percent' (%)
- `discount_value` — значення знижки (валідація: percent <= 100, всі >= 0)
- `note` — опціональна нотатка

**RLS**: Усі автентифіковані можуть читати (бейдж публічний); лише адмін (is_user_admin) може створювати/редагувати/видаляти.

**UI**:
- `<CertificateBadge userId={...} />` — золотистий градієнтний бейдж з іконкою Award + Tooltip; рендериться тільки коли `is_active && discount_value > 0`
- Розміщено absolute -bottom-1 -right-1 на аватарі в ProfileHeader
- Realtime-підписка на user_certificates для миттєвого оновлення бейджа
- Адмін-таб: live preview бейджа під час редагування, dirty-state кнопки «Зберегти», пошук

**Бізнес-правила**: Сертифікат видно публічно всім (стимулює інших користувачів). Без терміну дії — деактивується вручну адміном.
